# Roadmap

## MVP

- Local scanner for repo instructions and docs.
- Current-state capsule under about 700 estimated tokens where possible.
- Task-based read map.
- Caveman Brain local SQLite memory.
- Claude `/contextzero` skill workflow.

## Next

- Better import-chain analysis.
- Configurable thresholds.
- Richer memory extraction.
- Safer archive workflow with explicit approvals.
- Optional semantic search after the keyword MVP is proven.

## Not Planned For MVP

- SaaS.
- Cloud sync.
- Uploading repo contents.
- Guaranteed token savings claims.
